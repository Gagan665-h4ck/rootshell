# gs.dev — Hacker Portfolio

Personal portfolio of **Gagandeep Singh**, Cyber Security Analyst (VAPT).  
One-page dark hacker-themed site with interactive terminal, Matrix rain, network topology visualization, and cyberpunk city skyline.

## Features

- **Boot sequence** — Full-screen terminal-style kernel boot with auto-type
- **Interactive terminal** — 15+ shell commands with easter eggs
- **Matrix rain** — Animated falling characters background
- **Network topology** — Live particle graph with connection lines
- **Cyberpunk skyline** — Animated city silhouette at page bottom
- **Hackish grid** — Subtle green grid overlay with scanning line
- **Certification badges** — 10 cert cards with large images
- **CVE intelligence** — CVE-2026-25809 card with CVSS 9.1
- **Recon dashboard** — Stats cards (years active, CVE, certs, assessments)
- **Scroll reveal** — Fade-up animations on all sections
- **Cursor glow** — Green dot following mouse
- **Mobile responsive** — Adapts to all screen sizes

## Commands

| Command | Action |
|---------|--------|
| `whoami` | Display identity |
| `skills` | List skill set |
| `certs` | Show certifications |
| `contact` | Contact information |
| `clear` / `cls` | Clear terminal |
| `date` | Show current UTC time |
| `uptime` | Operator uptime |
| `neofetch` | System info |
| `ls` | Directory listing |
| `ll` | Detailed listing |
| `pwd` | Current path |
| `echo <text>` | Repeat text |
| `sudo <cmd>` | Run as sudo |
